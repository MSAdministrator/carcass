# carcass
carcass is a Python package to generate python package scaffolding based on best practices

## Template Structure

```
package_name
    README.md
    CONTRIBUTING.md
    LICENSE.md
    setup.py
    logger.yml
    .gitignore
    package_name
        __init__.py
        package_name.py
        connector.py
        utils
            __init__.py
            exceptions.py
            logger.py
```
